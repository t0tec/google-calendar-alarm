# Google Calendar Alarm
This application will trigger an alarm (play music out your music directory) when an event is about to start.
Also you can add/edit new (single) events to your calendar.

Using:

- Google API client
- Google OAuth client
- Google Calendar API
- Cron4J
- Joda Time
